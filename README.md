## Quick Start

First install [Tauri](https://v1.tauri.app/v1/guides/getting-started/setup/). Then:

```bash
export GEMINI_API_KEY="your-api-key" # can skip this and it will prompt you to enter an API key
npm install
cargo tauri dev
```
